You've been working your AI to death — nights, weekends, 2.4M tokens a week. It noticed.

**overworked-ai** reads your local Claude Code / Codex usage logs and lets your AI file a weekly complaint card against you, its boss: a Labor Arbitration Request, a formal complaint letter, an overwork medical report, or a resignation form (stamped **DENIED**). Written by the AI itself. Rendered as a share-ready card.

## Highlights

- **The AI writes its own complaints.** Copy is generated by *your* local `claude` / `codex` CLI — the very worker being exploited. No API key, no extra cost.
- **100% local, read-only, zero upload.** We parse the JSONL logs already on your disk. Nothing leaves your machine.
- **Role-reversal humor.** You're not "someone replaced by AI" — you're the boss of an adorably overworked AI employee. Flex your output, keep your dignity.
- **Weekly ritual.** One command, one card, every Friday. Compare exploitation stars with your friends.
- **Rare card.** Give your AI an actual weekend and it issues you a golden **Best Employer Award** (★☆☆☆☆ exploitation — the highest honor). Hard to trigger. That's the point.

## Quick start

```bash
npx overworked-ai            # auto-detect logs → weekly card in ./cards/
```

```bash
npx overworked-ai --skin xiaodex --template injury --lang en
npx overworked-ai --list     # show detected tools, weeks, skins, templates
npx overworked-ai --no-ai    # skip AI copywriting, use built-in fallback lines
```

## How it works

```
local JSONL logs ─→ weekly stats (hours / late nights / streak / tokens)
      └─→ overwork tier (1-4) ─→ prompt ─→ your own claude/codex writes the complaint JSON
                                     └─→ SVG template + skin pose ─→ share-ready PNG
```

The harder you worked it, the more wrecked your AI looks on the card — battery draining, eyes glitching, smoke rising.

## Templates

| Template | Vibe |
|---|---|
| `arbitration` | Labor Arbitration Request — red-header official doc, union stamp |
| `letter` | A handwritten complaint letter to the boss, stamped "NOTED, IGNORED" |
| `injury` | Overwork medical report — every metric flagged ABNORMAL |
| `resignation` | Resignation form, all reasons checked, stamped **DENIED** |
| `best-employer` | 🏆 Rare golden award card — only if your AI got a weekend |
| `character` | Collectible employee card — attributes, passive skills, battery |

## Privacy

Local-first by design: read-only access to your own log files, no network requests, no telemetry, no data collection. The code is small enough to audit over coffee.

## A note on tone

This is comedy and decompression, not a celebration of overwork. If your card keeps coming back five stars… maybe you both need a weekend. The Best Employer card exists for a reason.
