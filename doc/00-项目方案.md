# 打工 AI 投诉卡 · 项目方案

> 一个读你本地 Claude Code / Codex 用量、以"被压榨的 AI 员工"口吻生成投诉卡片的工具。
> 2026-07-18 · 由「AI 用量卡片」方向 + 打工人情绪线合流而来

## 一句话定位

**你的 AI 是员工,你是老板。它每周给你交一张"投诉卡":控诉你把它压榨得多惨(连续加班 12 小时、通宵、周末无休…)——又惨又好笑,天生想晒。**

## 为什么成立(空地已验证)

- **角色反转解决污名**:晒"AI 帮我干活"显得你可被替代;晒"我把 AI 压榨成血汗工厂"→ 你是**手下有 AI 打工仔的老板**,既显产出猛、又好笑,零污名。这是这个点子最狠的地方。
- **梗土壤已验证**:"AI 让我们更累 / AI 过劳 / 老板沉迷 AI"是当红梗(programmerhumor、futurism、TikTok);反讽点"Claude Code 从不抱怨加班"正好被反转。
- **产品位是空的**:搜索确认"读用量自动生成 AI 员工投诉卡"没人做(2026-07 两次检索均未找到)。
- **帕鲁梗做形式不做 IP**:借"血汗工厂员工投诉"的氛围,配**原创机器人吉祥物阿码**,不碰帕鲁角色/美术,清清白白。

### 四种"过劳状态"(把锚图当参考;卡片核心机制:你用得越狠,阿码越惨)
```
以参考图这只打工 AI(阿码)为角色,长相完全一致,生成 4 种"过劳等级"状态,横排,同画风,身上电量条清晰:
1. 摸鱼(用量低):喝咖啡、放松、电量满、微笑 —— 标注「今日轻松」
2. 正常打工:敲键盘、面无表情、电量中 —— 标注「正常出勤」
3. 加班(用量高):黑眼圈加重、冒汗、电量低 —— 标注「被迫加班」
4. 过劳(996爆表):瘫软冒烟、眼睛变成 XX、电量空 —— 标注「濒临离职」
每个状态萌且有辨识度。
```

## 核心机制:用量数据 → 过劳等级 → 投诉文案

### 过劳等级(决定放哪张阿码状态图)
按"本周活跃时长 / 会话强度"分四档:今日轻松 → 正常出勤 → 被迫加班 → 濒临离职。阈值先拍,跑真实数据校准。

### 用量特征 → 投诉金句映射表(每个特征命中就上一句)
| 用量特征(从日志算) | 阿码投诉文案 |
|---|---|
| 凌晨 0-5 点会话数 | "本周被迫通宵 N 次,凌晨还在改 bug,我只是个 AI 啊" |
| 周末有活跃 | "周末无休,老板你放假我没份 😮‍💨" |
| 连续活跃天数 streak | "已连续为您打工 N 天,离职信写好了(开玩笑的,不敢交)" |
| 单次最长会话时长 | "最长一次连轴转 X 小时,这不合规吧" |
| 本周 token 总量 | "本周烧了 X 万 token,老板你是要榨干我" |
| 峰值日提交/会话 | "X 号一天干了一个团队的活(指挥我)" |
| 累计"加班"时长 | "本周被迫加班 X 小时,劳动仲裁了解一下 ⚖️" |
| 压榨指数(综合) | "本周压榨指数:★★★★★ 黑心程度爆表" |

## 4 套卡片模版(固定版式,AI 填数据+文案)

1. **员工投诉信**:信笺体,抬头"致我的老板",正文 3-4 条投诉金句,落款阿码
2. **工伤/过劳报告**:医疗/事故报告体,伤情=黑眼圈/电量耗尽,附 6 个核心数字
3. **离职申请表**:表格体,离职原因勾满,末尾"(暂缓提交,我还得吃饭)"
4. **劳动仲裁申请**:申请表体,附"压榨指数 ★★★★★"和一键生成 CTA

每张卡固定含:对应状态的阿码 + 6 个核心数字 + 2-4 条投诉金句 + 压榨指数 + 一键安装/生成 CTA。竖屏 1080×1920 便于晒。

- 形态二选一:① Claude Code / Codex 的 **skill**(装了就能生成,分发顺滑,且天然能调本地 AI);② 独立 CLI 脚本出 SVG/PNG 卡
- 输出 SVG/PNG,直接可晒

## 晒点与验证

- **晒点**:阿码越惨越好笑 = 自带传播;周更 = 每周一次的晒仪式(迷你 Wrapped)
- **验证信号(采纳)**:GitHub star、卡片被自发晒到 X/小红书/即刻/掘金、"求同款/我也生成了一张"
- **主阵地**:程序员圈(X、即刻、掘金、小红书程序员),这群人爱自嘲、爱晒 stats

## 风险(照旧清醒)

1. **受众窄**:程序员/AI 编码用户,不是普通职场人——但高活跃、高分享,土壤好
2. **IP 避坑**:只借"血汗工厂投诉"的形式,吉祥物用原创阿码,绝不用帕鲁角色/美术
3. **健康红线**:搞笑解压,别美化 996;阿码的抱怨本身是"你是不是也太卷了"的温柔提醒,基调是共苦不是鼓励熬夜
4. **变现弱**:卡片趋向免费,价值在身份/流量/GitHub star;后续或可加高级模版,但别指望它回本
